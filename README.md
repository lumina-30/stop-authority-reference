# Stop Authority (Reference)

Stop Authority refers to the structural capacity of human institutions
to retain a final, effective refusal power
before irreversible autonomous execution.

This repository does not propose regulation, prohibition, or policy mandates.
It documents structural conditions under which Stop Authority may disappear.

It is descriptive and non-prescriptive.
